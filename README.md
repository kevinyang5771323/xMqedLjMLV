# 音乐后台管理系统 java867

## 项目概述

音乐后台管理系统是基于Springboot与Vue的前后端分离项目，致力于提供高效、便捷的音乐资源管理方案。

## 技术栈

- 前端：Vue
- 后端：Spring Boot
- 数据库：MySQL
- 构建工具：Node.js、Maven

## 功能模块

- 系统首页
  - 概览系统状态与基本信息
- 用户管理
  - 管理系统用户账号
- 歌手管理
  - 维护歌手信息与资料
- 歌单管理
  - 管理歌单内容与结构

## 系统角色

- 系统管理员：负责系统整体管理与维护
- 普通用户：使用系统提供的音乐管理功能

## 运行环境

- 开发工具：IDEA、VS Code
- 依赖环境：MySQL、Node.js、Maven

## 部署步骤

1. 准备开发环境，安装上述运行依赖
2. 使用Maven构建后端工程
3. 使用Node.js与相关依赖构建前端工程
4. 部署至合适的Java应用服务器，如Tomcat
5. 配置MySQL数据库，导入数据库文件

## 目录结构

```
项目根目录/
├── frontend/        # 前端项目目录
├── backend/         # 后端项目目录
└── database/        # 数据库文件目录
```

## 核心亮点

- 前后端分离，便于独立部署与维护
- 采用Vue与Spring Boot，确保高效稳定的系统性能
- 清晰的功能模块划分，便于快速上手和使用
